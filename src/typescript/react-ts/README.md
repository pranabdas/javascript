# React with TypeScript

Initialize project with `Vite`:
```console
npm create vite@latest react-ts -- --template react-ts
```

## Setup

Install dependencies:
```console
npm install
```

## Development

Start the development server:
```console
npm run dev
```

## Build

Build for production:
```console
npm run build
```

Preview production build:
```console
npm run preview
```
